My First Bootstrap Page

HTML

HyperText Markup Language

HTML is a markup language that allows you to create your own website and webpages. It uses tags and lines of code to create the output you want. This software is beginner friendly and compatible on Windows and MacOS.

CSS

Cascading Style Sheets

CSS works with HTML elements and allows you to add flare to your webpages. You can add designs or change the appearance using the various CSS properties. There are three kinds: inline, internal, and external.

Bootstrap

Bootstrap Version 4

Bootstrap makes web development easier. It adjusts the contents of your website to fit the size of the user's screen. This is so that your website looks consistent and nice across all devices.
